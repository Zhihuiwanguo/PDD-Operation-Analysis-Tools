# 拼多多运营分析工具 v1

## 项目目标
上传拼多多订单表、推广表、流水表、映射表后，自动完成经营分析与可视化，减少人工整理和重复计算。

## 第一版范围
- 整体经营概览
- 链接经营总表
- 产品经营分层表
- 百补 vs 日常对比
- 异常链接 / 异常规格 / 未映射清单
- 销量前五产品规格分析
- 结果导出 Excel / CSV

## 当前固定经营口径

### 订单有效性
- 有效订单：正常成交且售后状态不是退款成功
- 无效订单：已取消、未发货退款成功、已发货退款成功、已收货退款成功、其他退款成功
- 待确认：售后处理中，单独列示，不并入有效也不直接判无效

### 非经营剔除
- 差价补款商品：不纳入经营分析指标，归入非经营剔除项

### 推广费口径
- 链接经营盘推广费：推广汇总表字段“实际成交花费(元)”
- 店铺总盘推广费：推广账户每日流水表中的“现金支出”汇总

### 平台扣点
- 百补链接：商家实收 × 2.4%
- 非百补（日常）链接：用户实付 × 0.6%

### 毛利与ROI
- 订单侧估算毛利 = 商家实收 - 产品总成本 - 快递总成本 - 平台扣点
- 链接扣推广后贡献毛利 = 订单侧估算毛利 - 实际成交花费(元)
- 店铺扣推广后贡献毛利 = 订单侧估算毛利 - 店铺总盘推广费
- 链接实际ROI = 商家实收 ÷ 实际成交花费(元)
- 店铺整体实际ROI = 商家实收 ÷ 店铺总盘推广费
- 盈亏平衡ROI = 商家实收 ÷ 订单侧估算毛利

## 输入文件
系统第一版支持上传以下文件：
1. 标准产品主档表
2. 销售规格映射表
3. 店铺链接规格映射表
4. 拼多多原始订单表
5. 拼多多推广汇总表
6. 推广账户每日流水表
